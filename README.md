# Bitcoin Address Generator
 Script em Node.js com pacotes bit32, bit39 e bitcoinjs-lib. Ele gera uma chave privada em WIF e endereço público em P2PKH com foco em Bitcoin.

## Installation

```
npm install bip32
npm install bip39
npm install bitcoinjs-lib
```
